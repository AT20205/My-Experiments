# My Experiments

A collection of AI, LLM, Machine Learning, Data Science, and automation experiments built with modern Python tools and frameworks.

This repository contains hands-on projects, prototypes, notebooks, and utilities exploring:

- Large Language Models (LLMs)
- AI agents and workflows
- LangChain ecosystem
- RAG (Retrieval-Augmented Generation)
- Vector databases
- Gradio applications
- Data science and visualization
- Deep learning and transformers
- Local and cloud AI integrations
- Browser automation and scraping
